## How to Approach This Systematically

Work through these steps in order. Each one builds on the last — do not skip ahead.

**Step 1 — Project setup**
Create the Vite + Vue project. Confirm it runs. Delete the default boilerplate from `App.vue`.

**Step 2 — Render the raw data**
Add the `courses` ref. Use `v-for` to render each course name in a plain `<p>`. Confirm all 5 names appear before doing anything else.

**Step 3 — Build `CourseCard` in isolation**
Create `src/components/CourseCard.vue`. Hardcode fake values in the template first — make it look right. Then swap hardcoded values for props. Then use it in `App.vue` with real data.

**Step 4 — Summary bar**
Write the three `computed` counts. Display them. Verify the numbers match the data.

**Step 5 — Search**
Add `searchQuery` ref and a `filteredCourses` computed. Replace `courses.value` in the `v-for` with `filteredCourses`. Test it.

**Step 6 — Status filter buttons**
Add `activeFilter` ref. Update `filteredCourses` to apply both filters at once. Test every combination.

**Step 7 — Add Course form**
Add `showForm` and a `reactive` object for the form fields. Build the template — text inputs with `v-model`, radio buttons for status. Write `addCourse()` with validation. Test end to end.

**Step 8 — Style**
Write scoped CSS for `CourseCard.vue` and `App.vue`. Status badge must differ visually by value. Active filter button must differ from inactive ones.
