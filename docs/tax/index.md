# Canadian Individual Tax Preparation — Complete Course Index

**28-Week Professional Training Program for Ontario Tax Preparers**

!!! info "About This Course"
    This course takes a complete beginner from zero knowledge of Canadian tax to a fully operational Ontario tax preparer — ready for their first tax season. Each week builds on the previous. All content is written at expert level with realistic examples throughout.

    **Focus:** T1 personal income tax returns · Ontario residents · EFILE-certified professional practice

---

## How to Use This Index

Work through the weeks in order for the first pass. After completing the course, use this index as a quick-reference navigator — jump directly to any topic when a specific client situation arises.

The **most critical weeks** for a new preparer, in priority order: **2, 5, 3, 11, 12, 13, 25, 17, 16, 21.**

## Key Dates at a Glance

| Date | Event |
|---|---|
| January | T-slips begin arriving. CRA "What's New" published. Update rate card |
| **March 1** | **RRSP contribution deadline** (for prior tax year) |
| March 31 | T3 slips due from mutual funds and ETFs — wait for these before filing investment clients |
| **April 30** | **Individual filing deadline. All balances owing due** (including self-employed) |
| June 15 | Self-employed **filing** deadline (but payment was due April 30) |
| July | CCB and GST/HST credit recalculated based on prior year returns |
| October–November | Year-end planning calls with all clients |
| **December 12** | Last safe date to initiate securities donation transfers to charity |
| **December 29** | Last trading day for tax-loss harvesting (T+2 settlement) |
| **December 31** | TFSA contributions, FHSA contributions, charitable donations, dividend declarations |
